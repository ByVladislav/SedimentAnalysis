# SedimentAnalysis

![Скриншот программы](basic/win.png)

**SedimentAnalysis** – это программа для совместного анализа данных рентгено-флуоресцентного анализа (РФА) и Раман-спектроскопии. Она загружает базы данных RRUFF, идентифицирует минералы по спектрам, группирует совпадения по ID, а также может запрашивать экспертное заключение через API DeepSeek с построением круговой диаграммы распределения компонентов и выделением функциональных групп.

## Возможности

- **Загрузка данных** – выбор папки с файлами `x.txt` (РФА) и `r.txt` (Раман).
- **Работа с базами RRUFF** – автоматическая загрузка и кэширование спектров и минеральных формул из ZIP-архивов.
- **Идентификация минералов** – сравнение Раман-спектра с базой, группировка результатов по ID, подсчёт числа совпадений.
- **Визуализация** – три интерактивных графика: РФА-спектр, Раман-спектр, сравнение с эталонами.
- **Экспорт результатов** – полный отчёт в TXT с элементным составом, списком совпадений и заключением.
- **Интеграция с DeepSeek AI** – получение экспертного вердикта с распределением компонентов, функциональными группами и примечаниями.
- **Удобный интерфейс** – адаптирован под разрешение 2160×1440, все панели масштабируются.

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/yourusername/SedimentAnalysis.git
   cd SedimentAnalysis
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Поместите ZIP-архивы с базами RRUFF в соответствующие папки:
   - `raw_rruff/raman/` – для Раман-спектров;
   - `raw_rruff/xray/` – для РФА-данных.

4. В папку `elements/` положите CSV-файлы с описанием минералов. Файлы должны содержать колонки:
   `id`, `name`, `formula`, `strunz`, `elements`, `hyperlink`, `wavelength`, `status`, `comments`, `shift`.

5. Запустите программу:
   ```bash
   python main.py
   ```

## Использование

1. Нажмите **«Выбрать папку с данными»** и укажите папку, в которой находятся `x.txt` и `r.txt`.
2. Нажмите **«Запустить анализ»** – программа выполнит сравнение спектров с базами RRUFF и покажет результаты в левой панели.
3. Для получения экспертного заключения нажмите **«Получить вердикт AI»**. При первом использовании программа запросит API-ключ DeepSeek (если он не указан в `config.py`).
4. Результаты можно экспортировать в TXT с помощью кнопок **«Экспорт результатов»** или **«Все ID»** (выгружает все минералы из базы `elements`).
